# Narzędzia Informatyki - GitHub

Repozytorium utworzone w ramach ćwiczenia nr 13 z przedmiotu Narzędzia Informatyki.
Celem projektu jest zapoznanie się z platformą GitHub oraz sposobem
udostępniania kodu i materiałów informatycznych.

## Zawartość repozytorium
- przykładowy program w języku C# – gra „Wisielec”
- materiały informacyjne dotyczące e-learningu
- przykładowy branch

## Wykorzystane technologie
- C#
- Git / GitHub
