# Wisielec - C#

Prosty program konsolowy napisany w języku C#.
Gra polega na odgadywaniu hasła poprzez podawanie liter.

## Funkcje programu
- losowanie hasła
- obsługa wprowadzania liter
- liczenie pozostałych prób
- informowanie o wygranej lub przegranej

Kod zawiera komentarze wyjaśniające działanie poszczególnych fragmentów.
