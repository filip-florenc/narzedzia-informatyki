# Pozyskiwanie informacji i samouczenie w informatyce

Prezentacja przygotowana w ramach zajęć dydaktycznych.
Materiał dotyczy sposobów pozyskiwania informacji w informatyce
oraz znaczenia samodzielnego uczenia się w rozwoju kompetencji IT.

W prezentacji omówiono m.in.:
- wyszukiwanie informacji technicznych
- korzystanie z dokumentacji i forów
- platformy e-learningowe
- znaczenie samokształcenia w branży informatycznej

Materiał został udostępniony jako przykład treści informacyjnej
opublikowanej na platformie GitHub.
